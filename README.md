## <img src="URL_ИКОНКИ_МЕТОДА" width="50" height="50"> <img src="URL_ИКОНКИ_ПЛАТФОРМЫ" width="50" height="50"> Название раздела или URL

Краткое описание того, за что отвечает данный эндпоинт или блок информации.

Пример ответа:
```json
{
  "status": "success",
  "data": []
}
```

Имеет параметры:
- `parameter=value` — описание параметра.
